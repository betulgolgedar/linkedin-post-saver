# 🔖 LinkedIn Post Saver

LinkedIn akışındaki postları tek tıkla kaydet, etiketle ve organize et.

## Özellikler

- Postların altına otomatik "Kaydet" butonu eklenir
- Kaydedilen postları popup'tan görüntüle, ara ve filtrele
- Etiket ekleme ve silme
- Ücretsiz planda 20 post kaydı

## Kurulum

Chrome Web Store'dan indirin *(yakında)* veya [web sitemizi ziyaret edin](https://betulgolgedar.github.io/linkedin-post-saver/).
